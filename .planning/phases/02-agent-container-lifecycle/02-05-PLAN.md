<!-- From filepush-protocol.md -->

JSON request format:
```json
{
  "files": [
    { "path": "App.tsx", "content": "base64-encoded", "delete": false },
    { "path": "screens/Home.tsx", "content": "base64-encoded", "delete": false },
    { "path": "screens/Old.tsx", "content": "", "delete": true }
  ]
}
```

Response format:
```json
{ "written": ["App.tsx", "screens/Home.tsx"], "failed": [] }
```

Error codes:
- 403: invalid signature or expired URL
- 400: path traversal or bad request
- 404: sandbox not found on node
- 500: write failure
</interfaces>

```go
type Handler struct {
    hmacSecret []byte
    resolver   SandboxResolver
    logger     *slog.Logger
}

func NewHandler(hmacSecret []byte, resolver SandboxResolver, logger *slog.Logger) *Handler

func (h *Handler) ServeHTTP(w http.ResponseWriter, r *http.Request)
```

ServeHTTP implementation:
1. Reconstruct the full request URL (path + query)
2. Call auth.VerifyURL(fullURL, h.hmacSecret) -- returns error on invalid/expired
3. On VerifyURL error: return 403 with JSON error
4. Extract sandbox ID from URL path (/v1/sandboxes/{id}/files)
5. Call resolver.CodeDir(sandboxID) -- 404 if not found
6. Check Content-Type:
   - application/json: parse JSON body with files array, call WriteFiles(codeDir, files)
   - application/x-tar: call WriteTar(codeDir, r.Body)
   - else: 400 unsupported content type
7. Return 200 with WriteResult JSON

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| signed URL -> agent | HMAC validates that control plane authorized this request |
| file content -> filesystem | Untrusted content written to sandboxed bind mount |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-14 | Spoofing | handler.go | mitigate | HMAC-SHA256 signed URL validation via shared-go/auth.VerifyURL. 60s expiry. Constant-time comparison. |
| T-02-15 | Tampering | writer.go | mitigate | Path traversal prevention rejects ".." and absolute paths. Files written only to sandbox code directory. |
| T-02-16 | Elevation of Privilege | writer.go | mitigate | Symlinks in tar archives ignored. Files cannot escape the code directory. |
| T-02-17 | Denial of Service | handler.go | accept | No request body size limit in v1. Control plane is the bottleneck (307 redirect). Future: add configurable max body size. |

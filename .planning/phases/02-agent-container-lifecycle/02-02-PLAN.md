func PrepareBindMount(sandboxDir, appName string) (string, error):
- Validate appName contains no ".." or "/" (path traversal prevention)
- codePath = filepath.Join(sandboxDir, appName, "code")
- os.MkdirAll(codePath, 0755)
- os.Chown(codePath, 1000, 1000) -- UID/GID for node user inside container
- Return codePath

func int64Ptr(v int64) *int64 { return &v }

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03 | Elevation of Privilege | sandbox.go | mitigate | CapDrop ALL + CapAdd only CHOWN/SETUID/SETGID. no-new-privileges:true. Seccomp profile. PID limit 256. |
| T-02-04 | Tampering | PrepareBindMount | mitigate | Path traversal check rejects ".." and absolute paths in appName. Bind mount is read-write only for /app/code. |
| T-02-05 | Denial of Service | container resources | mitigate | Memory and CPU limits enforced via Docker cgroups. PID limit prevents fork bombs. |
| T-02-06 | Information Disclosure | container env | accept | APP_NAME and PORT are non-sensitive. Sensitive config not passed to containers. |

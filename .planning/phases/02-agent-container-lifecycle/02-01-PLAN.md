2. Run tests -- they must FAIL (RED phase). Commit: `test(02-01): add failing port allocator tests`.
3. Implement agent/internal/ports/allocator.go:
   - `type Allocator struct` with sync.Mutex, available map[int]bool, allocated map[int]bool, min/max int
   - `func NewAllocator(min, max int) *Allocator` -- initializes available map with all ports in range
   - `func (a *Allocator) Allocate() (int, error)` -- picks first available port, moves to allocated, returns it. Error if none available.
   - `func (a *Allocator) AllocateSpecific(port int) error` -- reserves specific port if available
   - `func (a *Allocator) Release(port int) error` -- moves from allocated back to available. Error if not allocated.
   - `func (a *Allocator) InUse() int` -- returns len(allocated)
   - `func (a *Allocator) Available() int` -- returns len(available)
   - All methods are goroutine-safe via mutex.
4. Run tests -- they must PASS (GREEN phase). Commit: `feat(02-01): implement port allocator`.

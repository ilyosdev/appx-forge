## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation |
|-----------|----------|-----------|-------------|------------|
| T-05-06 | I (Information Disclosure) | metrics.go | accept | /metrics exposes sandbox counts and node utilization (aggregate, non-sensitive operational data). Prometheus scraping is infrastructure-internal. |
| T-05-07 | E (Elevation of Privilege) | seccomp-default.json | mitigate | Seccomp whitelist blocks ptrace, mount, reboot, swapon. cap-drop ALL + minimal cap-add. no-new-privileges prevents SUID binaries. PID limit 256 prevents fork bombs. |
| T-05-08 | S (Spoofing) | config.go | accept | Agent token per node (SEC-05, already Phase 3) prevents cross-node impersonation |
| T-05-09 | T (Tampering) | file push | accept | HMAC-SHA256 signed URLs with 60s expiry (SEC-06, already shared-go/auth/hmac.go) prevent URL forgery |
</threat_model>

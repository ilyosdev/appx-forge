       Sections:
       - Identify: `forge sandbox list --state=failed`
       - Diagnose: `forge sandbox inspect <id>` to check failure_count, last event
       - Check logs: `forge sandbox logs <id> --tail=200`
       - Check events: `forge events --sandbox=<id>`

         c. Check node health: `forge node list` -- is the node healthy?

       - Prevention: Monitor failure_count, set alerts on state=failed count > 0

    3. **Create docs/runbooks/debug-stuck-container.md** (OPS-05):
       Sections:
       - Symptoms: Sandbox state=starting for >60s, or state=running but URL returns 502
       - Step 1: `forge sandbox inspect <id>` -- check state, node_id, container_id
       - Step 2: SSH to node, `docker ps -a | grep forge-<app_name>`
       - Step 3: `docker logs forge-<app_name> --tail=100`
       - Step 4: Check agent logs: `journalctl -u forge-agent -n 100`
       - Step 5: Check if container is actually running: `docker inspect forge-<app_name>`
       - Force remove: `docker rm -f forge-<app_name>`, then `forge sandbox restart <id>`
       - Root cause patterns: Metro crash loop, bind mount permission issue, port conflict, OOM kill

       **d. Create ForgeService** (NestJS wrapper):
       File: `backend/src/modules/deployment/forge.service.ts`
       ```typescript
       @Injectable()
       export class ForgeService {
         private client: ForgeClient;
         constructor(private config: ConfigService) {
           this.client = new ForgeClient({
             baseUrl: config.get('forge.controlUrl'),
             apiKey: config.get('forge.apiKey'),
           });
         }
         async createApp(appName, userId, image, resources) { return this.client.sandboxes.create({...}); }
         async deleteApp(sandboxId) { return this.client.sandboxes.destroy(sandboxId); }
         async getAppStatus(sandboxId) { return this.client.sandboxes.get(sandboxId); }
         async pushCode(sandboxId, files) { return this.client.sandboxes.pushFiles(sandboxId, files); }
         async restartApp(sandboxId) { return this.client.sandboxes.restart(sandboxId); }
         async getLogs(sandboxId, tail) { return this.client.sandboxes.logs(sandboxId, {tail}); }
       }
       ```

    })
}

| Control Plane -> Agent | Logs proxy crosses network boundary to agent |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Information Disclosure | handleListNodes | mitigate | Never return agent_token in node list response. Map store.Node to nodeResponse struct that omits token field. |
| T-06-02 | Denial of Service | handleGetLogs follow=true | mitigate | Set 60s timeout on proxy HTTP request to agent. Do not hold connection indefinitely. |
| T-06-03 | Elevation of Privilege | handleRemoveNode | mitigate | Verify node has 0 active sandboxes before allowing removal. Return 409 Conflict otherwise. |
| T-06-04 | Spoofing | handleGetLogs agent proxy | accept | Agent is on Tailscale mesh (trusted network). HTTP to agent is acceptable within mesh. |
</threat_model>

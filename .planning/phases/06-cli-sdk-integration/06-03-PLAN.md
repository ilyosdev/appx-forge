<interfaces>
<!-- Control plane API endpoints the CLI calls -->

Node list: GET /v1/nodes -> {nodes: Node[]}
Node register: POST /v1/nodes/register (security: []) -> {node_id, agent_token, heartbeat_interval_seconds}
Node drain: POST /v1/nodes/{id}/drain -> 200
Node remove: DELETE /v1/nodes/{id} -> 200 (or 409 if sandboxes exist)

Sandbox list: GET /v1/sandboxes?app_name=X&user_id=Y&state=Z&node_id=N&limit=L -> {sandboxes: Sandbox[]}
Sandbox get: GET /v1/sandboxes/{id} -> Sandbox  (id can be UUID or "app:name")
Sandbox destroy: DELETE /v1/sandboxes/{id} -> 202
Sandbox restart: POST /v1/sandboxes/{id}/restart -> 202
Sandbox logs: GET /v1/sandboxes/{id}/logs?tail=N&follow=bool -> text/plain

Routes list: GET /v1/routes -> {routes: Route[]}
Events list: GET /v1/events?sandbox_id=UUID&event_type=TYPE&limit=N -> {events: Event[]}
Healthcheck: GET /v1/healthz (security: []) -> {status, postgres, uptime_seconds}

All authenticated endpoints require Authorization: Bearer {token}.

Response types (from OpenAPI):
```
Node: {id, hostname, tailscale_ip, capacity_mb, used_mb, capacity_cpu, status, last_seen_at, registered_at, agent_version, running_sandboxes}
Sandbox: {id, app_name, user_id, node_id, container_id, image, state, url, resources, host_port, created_at, updated_at, last_active_at, failure_count, metadata}
Route: {app_name, sandbox_id, upstream, updated_at}
Error: {type, title, status, detail, instance} (RFC 7807)
```
</interfaces>

         ```go
         func monitorHeartbeats(ctx context.Context, store *store.Queries, cfg *config.Config, logger *slog.Logger) {
             ticker := time.NewTicker(time.Duration(cfg.HeartbeatIntervalSeconds) * time.Second)
             defer ticker.Stop()
             for {
                 select {
                 case <-ctx.Done():
                     return
                 case <-ticker.C:
                     // List all nodes, check last_seen_at
                     // If now - last_seen_at > interval * missThreshold: mark unhealthy
                     // If node was unhealthy and heartbeat resumed: mark healthy (done by heartbeat handler)
                 }
             }
         }
         ```
       - Start HTTP server with graceful shutdown (SIGINT/SIGTERM)
       - On shutdown: cancel context (stops heartbeat monitor), close pool, shutdown HTTP server with 10s timeout
       - Log startup message: "forge-control listening on :8080"

    3. Heartbeat monitor logic:
       - Every HeartbeatIntervalSeconds: list all nodes
       - For each node where status == "healthy":
         - If time.Since(last_seen_at) > heartbeatInterval * missThreshold: call UpdateNodeStatus(id, "unhealthy")
         - Log: "node %s marked unhealthy (missed %d heartbeats)"
       - This is NOT a reconciliation loop -- it only marks nodes unhealthy, never restarts containers

| T-01-05 | Tampering | commands table | mitigate | `FOR UPDATE SKIP LOCKED` prevents double-dispatch of commands |
| T-01-06 | Injection | store queries | mitigate | sqlc generates parameterized queries -- no string concatenation of user input into SQL |
| T-01-07 | Denial of Service | commands table | accept | No per-node command rate limit in schema; acceptable for internal-only API |

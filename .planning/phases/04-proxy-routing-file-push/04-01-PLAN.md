```http
GET http://localhost:2019/config/apps/http/servers/srv0/routes
```

From docs/contracts/proxy-routing.md -- Batch Update (500ms debounce):
1. Route change events collected in buffer.
2. First event starts 500ms timer.
3. Subsequent events within window added to buffer.
4. Timer fires or buffer reaches 50 -> flush all.
5. Batch flush uses POST /load with complete route configuration.
